# Proyecto de estadística

#### ** DESCRIPCIÓN **
Como aficionado a la NBA, me pareció interesante utilizar un dataset en el que se almacenaran datos estadísticos importantes de todos los jugadores de la NBA desde 1947, como se puede ver a lo largo del proyecto, se han encontrado algunos datos interesantes sobre las leyendas de la NBA, aunque con esta cantidad de datos, evidentemente, se podrían sacar muchas más conclusiones. 

#### ** RECOMENDACIÓN **
El paso del cuaderno desde Visual Studio Code a PDF no ha quedado lo claro que se desearía. Para una visión clara de los datos, se recomienda descargar el HTML y verlo en algún navegador. Igualmente se ha incluído en el repositorio el PDF por si se quiere ver desde ahí directamente.

#### ** ENLACE DATASET **
https://www.kaggle.com/datasets/sumitrodatta/nba-aba-baa-stats?select=Player+Season+Info.csv
-  Dentro de este dataset, se ha elegido concretamente el archivo PlayersTotal.csv, renombrado como nba_players.csv, que es donde se veían concretamente las estadísticas de los jugadores.
